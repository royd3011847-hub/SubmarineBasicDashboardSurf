# Web Based dashboard for TigerFish

## When will this program be used?
- when testing the connection of the sub
- When running a programe on the sub

## What will this project do?
- Starts, Stops, and Pauses the programe
- Gives rotation Data (roll, pitch, yaw)
- Gives location data (x, y, z)
- gives depth
- Times how long the programe has been running to the ms
- shows if the controller and submarine are connected
